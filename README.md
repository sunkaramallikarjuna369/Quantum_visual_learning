# Quantum Visual Learning Platform

## Overview
An interactive, comprehensive tutorial for beginners to learn quantum computing through visualization and hands-on experimentation.

## Features

### 1. Interactive Circuit Simulator
- Drag-and-drop quantum gates (H, X, Y, Z, CNOT, etc.)
- Real-time circuit execution visualization
- Probability distribution display
- Multiple qubit support

### 2. Bloch Sphere Visualization
- 3D interactive sphere showing qubit states
- Real-time rotation as gates are applied
- Color-coded states for easy understanding

### 3. State Vector Display
- Live quantum state representation
- Complex amplitude visualization
- Phase information display

### 4. Quantum Algorithms
- Deutsch-Jozsa Algorithm
- Grover's Search Algorithm
- Bell State Generator
- Phase Estimation

### 5. Step-by-Step Tutorials
- Fundamental concepts explained simply
- Interactive examples
- Quiz sections for reinforcement

## Getting Started
Simply open `index.html` in a modern web browser.

## Requirements
- Modern web browser with WebGL support
- JavaScript enabled
