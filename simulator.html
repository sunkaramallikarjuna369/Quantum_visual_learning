🚀 How to Use

1

Pick Qubits

Choose 1-4 qubits using dropdown

2

Select Qubit

Pick which qubit: q[0], q[1], q[2], q[3]

3

Add Gates

Click gate button - adds to that qubit

4

Run Circuit

Click green RUN button!

💡 Building Circuits

  • Set target qubit with "Add to:" dropdown
  • Each gate click = adds 1 gate
  • Click gate on circuit to remove it
  • Click "Clear" to reset
