🚀 Quick Start Guide

Follow these simple steps to build and run your first quantum circuit:

1

Select Qubits

Choose how many qubits (1-4) using the dropdown menu. More qubits = more complex states.

2

Choose Target Qubit

Select which qubit (q[0], q[1], etc.) to add the gate to using the dropdown.

3

Add Gates

Click gate buttons to add them to your selected qubit. Gates appear on qubit wires from left to right.

4

Run & Remove

Click "Run Circuit" to simulate. Click any gate on wire to remove it. Or click "Clear" to reset.

💡 Tips

  • Use the "Target Qubit" dropdown to choose which qubit gets the gate
  • CNOT, CZ, and SWAP affect pairs of qubits
  • Try the example circuits to learn common patterns!
  • The Bloch sphere shows qubit q[0]'s quantum state

Superposition

q[0]: H

Creates equal superposition from |0⟩. Results in 50% chance of |0⟩ or |1⟩.

Result: 50% |0⟩, 50% |1⟩

Pauli-X (NOT Gate)

q[0]: X

Flips |0⟩ to |1⟩. The quantum equivalent of a classical NOT gate.

Result: 100% |1⟩

Bell State

q[0]: H → ●
q[1]: ⊕

Creates maximally entangled pair. The foundation of quantum teleportation.

Result: 50% |00⟩, 50% |11⟩

GHZ State

q[0]: H → ● → ●
q[1]: ⊕
q[2]: ⊕

Three-qubit entanglement. All-or-nothing correlation.

Result: 50% |000⟩, 50% |111⟩

Deutsch Circuit

q[0]: X, H → ●
q[1]: H

Part of Deutsch-Jozsa algorithm. Shows quantum parallelism.

Result: Shows interference pattern

Phase Kickback

q[0]: X, H
q[1]: H → Z

Demonstrates phase kickback - key to many quantum algorithms.

Result: |11⟩ with phase

📋 Common Circuit Patterns

Creating Superposition

Apply Hadamard (H) to |0⟩ to create superposition:

Before: |0⟩
Gate: H
After: (|0⟩ + |1⟩)/√2

Creating Entanglement (Bell State)

H on qubit 0, then CNOT from qubit 0 to qubit 1:

q[0]: ──[H]──●──
q[1]: ──────⊕──

Result: (|00⟩ + |11⟩)/√2

Measurement Pattern

Always put measurement gates at the END of your circuit:

q[0]: ──[H]──[M]──
q[1]: ──[CNOT]──[M]──

⚠️ Important Notes

  • CNOT uses qubit 0 as control, qubit 1 as target
  • Multi-qubit gates create correlations between qubits
  • The Bloch sphere only shows qubit 0's state (for single qubit view)
  • More "shots" = more accurate probability distribution
