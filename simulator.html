🚀 Quick Start

1

Select Qubits

Choose 1-4 qubits (shows q[0], q[1], q[2], q[3])

2

Choose Qubit

Select which qubit (q[0], q[1], q[2], q[3]) to add gate

3

Click Gate

Click a gate button - it adds to selected qubit

4

Run & See

Click "Run Circuit" and view results

💡 How to Build Circuits

  • Step 1: Pick number of qubits (1-4)
  • Step 2: Choose target qubit from "Add to" dropdown
  • Step 3: Click any gate button to add it
  • Step 4: Click "▶ Run Circuit" to simulate
  • Example - Bell State: Set to 2 qubits, add H to q[0], change to q[1], add CNOT
